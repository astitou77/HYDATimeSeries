HYDAT Database:

https://www.canada.ca/en/environment-climate-change/services/water-overview/quantity/monitoring/survey/data-products-services/national-archive-hydat.html

This tool helps to visualize a large national hydrological database using python to search, filter, and narrow data identification per station.

The tools also allows to generate SARIMA (statistical method) factors to generate reliable future predictive values for water levels, water flow, and sediment contents.
